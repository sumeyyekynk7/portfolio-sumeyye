# Sümeyye Kaynak Portfolio

Modern, responsive ve koyu temalı kişisel portfolio sitesi. React, Vite, Tailwind CSS ve Framer Motion ile geliştirildi.

## İçerik

- Hero
- Hakkımda
- Deneyim
- Projeler
- Yetenekler
- Eğitim
- Sertifikalar
- İletişim

## Kurulum

```bash
npm install
```

## Geliştirme

```bash
npm run dev
```

Vite varsayılan olarak projeyi `http://localhost:5173` adresinde çalıştırır.

## Build

```bash
npm run build
```

## Önizleme

```bash
npm run preview
```
